        Web3D可视化面试题和模板(WebGL/Three.js)

        # Web3D可视化面试题和模板(WebGL/Three.js)

        下面给大家简单介绍下Web3D可视化面试(WebGL、Three.js)中可能会遇到的问题。

        不过有一点要提前声明下，Web3D可视化起步发展中，并不像普通前端有那么多固定的面试模板或留言，面试形式五花八门，也不一定就是面试具体知识点。大家不用刻意去记忆下面的面试题，面试时根据自己掌握的知识，结合项目，随机应变回答即可。

        大家要做的是，根据我写的面试题，判断下，你自身的水平，查漏补缺，慢慢提升自身水平即可。

        # 面试大方向指导

        不同公司面试方式五花八门，不过大体上分为两大类。

        如果中小公司，刚刚接触这块，公司没人了解，一般倾向于让你展示作品或履历，而不是具体问题，毕竟他们也不会。

        如果对方公司有完整WebGL程序员团队，一般会问具体threejs、webgl、Cesium等具体知识点，如果是实习生，一般要求低点，如果是中高级程序员，除了简单threejs知识点，还会问数学几何计算、shader、WebGL等底层知识点。

        还有一些公司会给你一个模型场景，让你回去，自由发挥实现一个特定效果。

        # 作品问题

        你可以在我系统课程中一些案例基础上，丰富下，自己实现一些效果，或者放在个人博客上。(有一点，如果你的面试官刚好接触过我课程，你需要想好怎么回复，比如直接说明学过我课程，自己增加了点功能即可)。

        去模型网 (opens new window)等任何网站购买一些原创模型，然后自己写代码，这样就省去建模时间，然后直接套用课程用功能代码就行，模型虽然不同，但是代码没什么区别。

        当然了如果你有公司履历就用不着这样了，直接展示公司做的案例即可。

        # 前端问题面试

        一般前端问题面试，相比普通前端，会问的少点，毕竟主要还是面试3d这块。

        如果你原来就是前端，那么和以前面试一样就行。

        如果你是非前端转过来的，那么对你普通前端技术栈可能没有那么熟悉，但是也尽量去了解熟悉下。如果问到某块知识点你熟悉吗，如果不熟悉，直接回答否就行或者说知道点，但是很少用，这样面试官也不会深入问了。这样面试官就会重点面试web3d这块。

        # threejs基础问题

        一般会问提一些web3d可视化开发常见的基础问题，比如three.js、模型软件之类的。

        # 你都了解什么建模软件

        下面我写的非常多，但是你不用去记忆，其实能知道常用的Blender就行了，或者多说点美术常用的3dmax、c4d之类的

        美术相关：blender、3dmax、c4d、maya 机械相关：SolidWorks、CATIA、AutoCAD、UG 建筑相关：Revit、SketchUp 工业设计相关：Rhino

        # 介绍下模型几何体的position、uv、index、normal等属性

        1. position：顶点位置坐标，描述模型的几何形状
        2. uv：顶点UV纹理坐标，与顶点位置一一对应，用来表示贴图与模型的映射关系
        3. index：几何体顶点索引
        4. normal：几何体顶点法线

        更具体学习参考threejs中文网电子书章节2 (opens new window)和章节5 (opens new window)

        # 渲染循环问题

        具体问题：three.js动画渲染循环中，一般需要执行WebGL渲染器那个方法

        答案：一般需要执行WebGL渲染器WebGLRenderer的.render()渲染方法

        function render() {
    renderer.render(scene, camera); 
    requestAnimationFrame(render);
}

        # Group和Object3D区别

        在threejs的语法中Object3D是的Group父类,但是具体使用，都差不多，Group更语义化一些，可以用来表示一个组对象，可以包含mesh、group、line等子对象，构成层级模型。

        # 你了解threejs哪些材质

        整体来说，有点材质、精灵材质、线材质、网格材质等等，这些材质分别和点模型Points、精灵模型Sprite、线模型Line、网格模型Mesh相对应。

        其中网格材质比较丰富，比较常用，比如不受光照影响的基础网格模型MeshBasicMaterial，受到光照影响的漫反射网格材质MeshLambertMaterial、高光网格材质MeshPhongMaterial、PBR相关材质MeshStandardMaterial和MeshPhysicalMaterial

        相关链接： http://www.webgl3d.cn/pages/2da02c/ http://www.webgl3d.cn/pages/24c407/ http://www.webgl3d.cn/pages/56b66b/

        tween.js是JavaScript动画库，主要用于补间动画。在threejs项目中，也可以借助tween.js辅助threejs实现动画效果。

        模型世界坐标：.getWorldPosition()方法

        # 了解gltf文件格式吗

        gltf是Web3D可视化开发中，非常常用的一种三维模型文件格式，就像图像常用的jpg或png格式一样。

        一般实际开发，通过Blender构建三维模型，然后导出gltf文件，threejs在通过代码加载gltf模型。

        .glb文件是gltf格式的二进制文件。

        相关内容参考threejs中文网电子书章节6：http://www.webgl3d.cn/pages/f24993/

        具体参考：http://www.webgl3d.cn/pages/e1e75d/

        实际回复，你可以把你大概熟悉的说下，比如outline高亮发光描边，比如bloom发光、比如抗锯齿的相关后处理、比如gama矫正的相关后处理。

        # 一个三维场景有哪些标注标签方式

        1. threejs自身的模型对象，比如精灵模型Sprite
        2. CSS2和CSS3渲染器，渲染HTML标签标注三维场景

        参考：http://www.webgl3d.cn/pages/428714/

        # 怎么通过threejs代码设置三维场景的阴影

        记不太清，可以大概描述即可，一般就是递归遍历整个三维场景，设置所有模型可以产生阴影和接收阴影效果。

        然后通过阴影属性.shadow.camera，设置好阴影渲染范围。

        最后不要忘了，渲染器允许阴影渲染、光源允许阴影计算。

        具体参考：http://www.webgl3d.cn/pages/1dbe9d/

        1..castShadow设置产生阴影的模型对象 2..castShadow设置产生阴影的光源对象 3..receiveShadow设置接收阴影效果的模型 4..shadowMap.enabledWebGl渲染器允许阴影渲染 5..shadow.camera设置光源阴影渲染范围

        # 你知道怎么鼠标单击拾取选中模型吗？

        一般就是借助射线拾取API Raycaster实现。

        1. 根据鼠标点击事件对象，获得鼠标在canvas画布上的屏幕坐标(单位像素px)
        2. 把屏幕坐标转化为WebGL标准设备坐标
        3. .setFromCamera()根据标准设备坐标和相机参数计算射线
        4. 最后通过Raycaster实现射线拾取模型计算

        具体流程：http://www.webgl3d.cn/pages/f96af1/

        一般长期做这块，一些基本的数学几何计算还是要了解，更多内容，可以查看threejs中文网电子书或系统课程介绍。

        # 图形学方面，你了解哪些常用几何变换的矩阵

        模型矩阵：旋转矩阵、缩放矩阵、平移矩阵

        相机相关的视图矩阵、投影矩阵

        # threejs控制模型姿态的方式有哪些，简单说说

        1. 角度属性.rotation(欧拉角)
        2. 四元数属性.quaternion
        3. rotateX、rotateY、rotateZ等旋转方法，可以改变角度属性.rotation
        4. 旋转矩阵

        # 四元数和欧拉角有什么区别

        四元数和欧拉角，都可以控制物体的姿态角度变化。

        欧拉角存在自锁现象，四元数不存在自锁现象。

        # 向量点乘几何含义

        简单回复就是两个向量长度乘以夹角余弦值，更具体参考：http://www.webgl3d.cn/pages/44b0ba/

        # 怎么计算两个向量的夹角

        两个向量归一化，然后dot点乘即可

        # 向量叉乘几何含义

        1. 两个向量叉乘，得到的新向量垂直于两个向量
        2. 相乘结果向量长度是：两个向量长度乘以夹角正弦值

        更多代码参考：http://www.webgl3d.cn/pages/4558d6/

        # 怎么计算一个三角形平面的法线

        三角形随便选择两条边，生成两个向量。

        两个向量叉乘，得到一个垂直于三角形向量，结果向量归一化，就是三角形法线方向。

        # 判断点是否在扇形区域内

        一般一些游戏相关功能，可能会问到。

        简单回复就是，利用向量点乘，判断点是否在扇形区域内。利用向量点乘计算夹角，与扇形角度比较，然后计算点与扇形圆形距离与扇形半径比较即可。

        更多代码参考：http://www.webgl3d.cn/pages/feb38e/

        # 底层webgl或threejs shader问题

        一般有webgl专门团队公司，在北上广深一线城市，20k朝上webgl程序员，一般还是要求shader这块的，除非特殊情况，对方公司没人懂，招聘前端，懂threjs优先，可能不要求shader。

        # 介绍下渲染管线的原理

        形象点说，WebGL渲染管线就像工厂的流水线一样，提供了不同的功能单元，比如顶点着色器、图元装配、光栅器、片元着色器、深度测试、模板测试等不同功能单元，渲染管线通过这些功能单元，可以处理三维场景的顶点、光源、纹理图像等数据，渲染三维场景，在Canvas画布上输出渲染结果。

        1. 顶点着色器：执行顶点着色器代码
        2. 片元着色器：执行片元着色器代码

        更多关于webgl渲染管线详细的介绍，可以参考threejs中文网 (opens new window)WebGL入门讲解即可。

        # 原生WebGL你怎么创建一个矩形

        可以用两个三角形拼接出来一个矩形，三个顶点表示一个三角形，两个三角形需要6个顶点坐标。

        两个三角形有个两个顶点坐标是共享的，也可以借助顶点索引，这样创建一个顶点坐标，然后利用索引重复利用其中两个顶点即可。

        参考：http://www.webgl3d.cn/WebGL/

        http://www.webgl3d.cn/WebGL/

        根据自身情况具体说明就行

        # 你写过webgl或threejs的shader吗

        根据自身情况回复，如果写过具体的说明，如果没写过的复杂案例，可以回复大概入门，能写一些简单。

        学习的话，参考系统课程里面threejs shader或原生webgl或原生webpgu的讲解。

        # 通过threejs材质那个方法可以修改已有材质的shader

        所有threejs材质有一个父类Material，Material具有.onBeforeCompile()方法，通过.onBeforeCompile()方法可以修改、增删threejs材质默认的shader，比如你可以修改Lambert网格材质的着色器代码，可以修改PBR材质的着色器代码。

        具体参考系统课程中threejs shader部分的讲解

        # threejs shader实现飞线动画

        飞线轨迹是通过线条几何体的顶点坐标构成，可以使用点模型Points渲染飞线轨迹，然后通过顶点着色器gl_PointSize控制飞线点的像素大小，按照飞线顶点顺序，控制点像素大小渐变即可。

        具体在地球飞线案例中有讲解

        # threejs shader设置模型颜色渐变

        在片元着色器GLSL代码中，通过片元坐标vPosition控制颜色值，比如让片元的颜色值gl_FragColor与vPosition的z坐标相关。

        vec3 gradient = mix(vec3(0.0,0.1,0.1), vec3(0.0,1.0,1.0), vPosition.z/550.0);
outgoingLight = outgoingLight*gradient;
gl_FragColor = vec4( outgoingLight, diffuseColor.a );

        # threejs shader实现直线扫光效果

        .onBeforeCompile()批量修改场景中所有模型的材质shader。

        比如通过vPosition的x坐标分量，在x1~x2范围内，控制顶点颜色值，实现直线扫光效果。如果希望扫光运动起来，可以加入一个uniform时间变量t，控制x1~x2范围随时间变化，实现直线扫光动画效果。

        # threejs shader实现直线波动光圈扫光效果

        比如通过vPosition的x、y坐标分量控制，计算三角函数计算x、y坐标与某个圆心的半径长度，设置半径r1~r2范围，控制顶点颜色值，比如设置白色。如果希望扫光运动起来，可以加入一个uniform时间变量t，控制r1~r2范围随时间变化，实现直线光圈动画效果。
